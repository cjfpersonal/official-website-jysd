# official-website-jysd

个人技术学习，主要包括typescript、vue、pinia模块

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```


### 技术
- typescript
- vue
- pinia
